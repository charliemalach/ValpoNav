ValpoNav




About

Interactive Map Navigation

ValpoNav was designed to help students, staff, and the public navigate Valparaiso University's campus effectively and in depth. To achieve this, ValpoNav uses an interactive map that allows users to navigate, zoom, and click on locations.

Building Searching

With ValpoNav, you are able to search for buildings on Valparaiso University's campus, and locations off campus that apply to the One Card program. This is useful if you are unaware of a building's general location.

Map Features

The map used for ValpoNav has many functions including clickable location markers, scrolling zoom, a toolbar for geolocation to precisely find your location, and a home button to return to the center if you get lost.

Navigation Menu

The navigation menu for the map can be found in the top left. When clicked, a side menu will extend allowing you to select and deselect buildings within the provided categories making navigation easier.

Building Descriptions

Have you ever wondered what goes on in a certain building? This can be answered with the more info feature found when clicking on a building marker. After clicking "more info" you will be redirected to information regarding the selected building.

One Card Locations

Valparaiso University has partnered with several companies to use your one card for purchases and also providing discounts. These companies can be found in the off campus categegory. For more info, click on the marker and check out "more info".
