Frequently Asked Questions

Can I select multiple buildings?

Yes, this can be done within the menu. For more info on how to select and deselect certain buildings, click here: Help

Can I find more info about buildings?

Yes, this can found by clicking the building marker and building labels. For more info on finding more information about buildings, click here: Help

Can I locate where I am?

Yes, this can be found by using the geolocation button on the map toolbar. For more info on finding your location on the map, click here: Help

Can I select multiple buildings from multiple categories?

Yes, this can be done within the menu and by selecting multiple buildings from multiple categories. For more info on how to select multiple buildings from different categories, click here: Help

Can I use my one card at the Off Campus locations?

Yes, all students are able to use their one card at all Off Campus locations listed on this map. For more info on how what buildings you can use your one card at, click here: More Info

Can I see individual classrooms?

Yes, these can be found in the floor plans in "More Info" on a buildings marker. For more info on classroom locations for a building, click here: Help
